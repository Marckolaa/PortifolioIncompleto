LearnMate

LearnMate é uma plataforma educacional desenvolvida com o objetivo de auxiliar os jovens de Portugal e do mundo na preparação para os exames de acesso ao ensino superior. A iniciativa surgiu para atender às necessidades dos estudantes que enfrentam dificuldades específicas nas provas, oferecendo uma experiência de estudo personalizada.

Funcionalidades principais:
Testes Personalizados: Os estudantes podem realizar testes sob medida com base em suas necessidades, possibilitando um aprendizado mais eficiente e focado nas áreas que precisam ser reforçadas.

Análise de Desempenho: A plataforma oferece um painel de controle onde os estudantes podem monitorar o progresso e obter feedback imediato sobre seus resultados.

Recursos Educacionais Gratuitos: LearnMate também conta com uma seção de recursos educativos gratuitos para os usuários que se registrarem.

Tecnologias utilizadas:
Front-end: React.js

Back-end: Node.js

Banco de dados: MongoDB

Autenticação: Auth0

Pagamentos: Stripe
